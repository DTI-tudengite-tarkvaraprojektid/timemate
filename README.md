# Timemate / Üliõpilase ajakasutus

Meie juurde tuli klient probleemiga, et pole teada kui palju õpilased reaalselt aega kulutavad õppetegevusele digitehnoloogia instituudis ja kui ühtlaselt see jaguneb semestri vältel. EAP on teoorias 26 tundi õppetööd aga siiani on seda arvestatud umbes tunde järgi. Seepärast 
ongi loodava rakenduse eesmärk jälgida õpilaste tegelikku ajakasutust ja anda õpilastele võimalus võrrelda teiste õpilaste keskmisega.
